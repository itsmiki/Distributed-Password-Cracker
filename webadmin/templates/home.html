Distributed Password Cracker Dashboard


This dashboard provides an interface for managing distributed password-cracking operations powered by Hashcat. The tool allows users to launch various types of attacks across multiple connected clients, harnessing their combined processing power for faster results.

Functionalities:
  • Bruteforce Attack: Perform a comprehensive, systematic attempt to crack a password by trying all possible combinations of characters until the correct one is found.
  • Dictionary Attack: Leverage a predefined list of potential passwords (dictionary file) to guess the correct password quickly if it exists within the list.
  • Dictionary Attack with Mask: Combine the power of dictionary attacks with specific custom patterns (masks) to optimize the cracking process by limiting the range of guesses, thus improving efficiency.
  • Identify Hash: Automatically detect and identify the type of hash (e.g., MD5, SHA-256, Whirlpool) from the provided input. This is a crucial first step in selecting the appropriate attack strategy and optimizing performance.

This platform enables the easy distribution and coordination of these attacks across multiple clients, making the process more efficient and scalable.
